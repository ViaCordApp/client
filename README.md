# ViaCord

A revival of the classic 2015 Discord client, connected to [app.viacord.org](https://app.viacord.org).

## Download

Grab the latest release from the [Releases](../../releases) page and extract the zip. No installer needed — just run `Discord.exe`.

## Requirements

- Windows (7 or later)
- The zip extracted somewhere you have write access (e.g. your Desktop or Documents)

## Usage

1. Download and extract the zip from Releases
2. Run `Discord.exe`
3. Create an account or log in at [app.viacord.org](https://app.viacord.org)

The client comes pre-configured to connect to the ViaCord server — no setup needed.

## What is this?

ViaCord is a self-hosted recreation of the Discord experience as it was in mid-2015, running the original Discord desktop client (v0.0.274) against our own backend. Features include:

- Text channels and DMs
- Friend requests and relationships
- Server creation and invites
- Voice chat (desktop and browser)
- The original 2015 Discord UI


the source in the /app folder is the decompiled app.asar from the vanilla client
